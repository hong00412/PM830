★ 기본 selector

  • tag selector

    css에서 tag를 지정할 때는 아무 기호 없이 tag name으로 작성

  • class selector

    tag에 class를 지정하였을 경우 해당 class name 앞에 '.'을 붙여 작성

  • id selector

    tag에 id를 지정하였을 경우 해당 id name 앞에 '#'을 붙여 사용

★ attr selector

  • attr selector

    tag에 있는 attr을 지정해서 사용하고 싶은 경우 tag name 뒤에 '[attr="value"]' 작성
    ex) input[type="text"]

  • string selector

    tag에 있는 attr 중 일부 string만 활용하여 지정하고 싶을 경우 1. tag[attr='value'] = tag의 attr에 적용되어 있는 value를 지정
    2. tag[attr$='value'] = tag의 attr에 적용되어 있는 value로 끝나는 것을 모두 지정
    3. tag[attr^='value'] = tag의 attr에 적용되어 있는 value로 시작하는 것을 모두 지정
    4. tag[attr*='value'] = tag의 attr에 적용되어 있는 value가 들어있는 모든 것을 지정

★ 기호로 구분하는 selector

  • 직계 selector

    부모 tag 바로 직계 자식의 tag를 잡고 싶을 때는 '>' 사용
    ex) div > h2 = div 바로 밑에 있는 직계 h2

  • 바로 뒤에 있는 tag selector

    바로 뒤에 있는 tag를 잡고 싶을 때는 '+' 사용(같은 라인 tag만 가능, 자식이나 부모tag를 잡을 수 없음)
    ex) input[type='checkbox'] + label = input 중 checkbox 바로 뒤에 있는 label을 잡는 것

  • 뒤에 있는 모든 tag selector

    기준이 되는 tag 뒤에 있는 모든 tag를 잡고 싶을 때는 '~' 사용(같은 라인 tag만 가능, 자식이나 부모tag를 잡을 수 없음)
    ex) div ~ p = div 뒤에 있는 모든 p를 잡는 것

★ 상태 selector

  • status selector

    tag가 어떤 상태일 때를 잡고 싶은 경우 1. input[type='checkbox']:checked = input checkbox가 check 상태인 경우
       (checkbox만 가능, radio는 다른 radio를 선택하지 않는 이상 선택을 해제할 수 없음)
    2. input[type='text']:enabled = input text가 수정 가능한 상태일 경우
    3. input[type='password']:disabled = input password가 수정 불가능한 상태일 경우

    etc - input의 placeholder의 css 속성을 바꾸고 싶은 경우엔 '::placeholder' 사용
          ex) input[type='text']::placeholder

★ nth selector

  • nth-child

    선택하고 싶은 tag의 부모 tag 기준으로 몇 번째인지를 찾아 선택하는 selector 1. first-child = 부모 tag 중 첫번째 자식 tag
       ex) ul li:first-child = ul을 기준으로 첫번째에 있는 li tag
    2. last-child = 부모 tag 중 마지막 자식 tag
       ex) ul li:last-child = ul을 기준으로 마지막에 있는 li tag
    3. nth-child(numb) = numb에 본인이 선택하고 싶은 순서의 숫자를 작성
       ex) ul li:nth-child(3) = ul을 기준으로 세번째에 있는 li tag

  • nth-of-type

    같은 line에 있는 형제 tag 중 같은 tag를 기준으로 그 중 몇 번째인지를 찾아 선택하는 selector 1. first-of-type = 형제 tag 중 같은 tag 기준으로 가장 첫번째인 tag
       ex) div p:first-of-type = div 안에 있는 모든 직계 p tag를 기준으로 가장 첫번째에 있는 p tag
    2. last-of-type = 형제 tag 중 같은 tag 기준으로 가장 마지막인 tag
       ex) div p:last-of-type = div 안에 있는 모든 직계 p tag를 기준으로 가장 마지막에 있는 p tag
    3. nth-of-type(numb) = numb에 본인이 선택하고 싶은 순서의 숫자를 작성
       ex) div p:nth-of-type(3) = div 안에 있는 모든 직계 p tag를 기준으로 3번째에 있는 p tag

★ not selector

  • not selector

    선택하고 싶지 않은 tag를 잡을 때 사용하는 selector 1. attr 선택자로 뺄 때 -> input:not([type="text"])
    2. class 선택자로 뺄 때 -> input:not(.class)
    3. id 선택자로 뺄 때 -> input:not(#id)
    4. nth 속성 선택자로 뺄 때 -> input:not(:first-child)

★ 기타 selector

  • 첫 글자만 선택하는 selector

    text중 가장 첫 글자를 잡고 싶을 때는 'tag:first-letter' 사용
    ex) h1:first-letter = h1의 text중 가장 앞 글자만 선택

  • 첫 줄만 선택하는 selector

    text 문장 중 가장 첫 줄을 잡고 싶을 때는 'tag:first-line' 사용
    ex) p:first-line = p의 text중 첫 줄만 선택(반응형에 취약)
